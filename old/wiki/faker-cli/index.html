    --locale

    Combining --locale option with --verbose will output translated values using the selected locale.

    • shell

    • output 

    lookup names --verbose --locale uk

    Faker()
├── artist
│   └── names() // => Сандро Ботічеллі
├── greekPhilosophers
│   └── names() // => Pythagoras
├── heroes
│   └── names() // => Tazar
├── heroesOfTheStorm
│   └── classNames() // => Support
└── pokemon
    └── names() // => Onix
    info

    Notice how some values are not translated because of a partially-localized uk dictionary.
    See also Faker Configuration - Locale for more information about generating values in another language.
